# Agentic Skills & Specialized Workflows

To maintain high development standards, use the following specialized skills available in `.agents/skills/`:

1.  **arxiv-reproduce**: Autonomously reproduces quantum computing arXiv papers using TensorCircuit-NG. Creates standardized repository structures, generates meta.yaml, writes/runs JAX-accelerated code, and strictly enforces code quality before saving final figures.
2.  **code-reviewer**: Autonomously reviews and refactors code for mathematical correctness, high-performance JAX-native patterns, backend agnosticity, and strict adherence to software engineering best practices (DRY, static analysis) with minimal architectural changes.
3.  **demo-generator**: Transforms a TensorCircuit-NG script into an interactive, sleek, and high-performance GUI application using Streamlit with interactive widgets and real-time visualizations.
4.  **meta-explorer**: Autonomously explores the space of quantum circuits and optimization strategies to solve specific physical problems (VQE, QAOA, QML, etc.) using budget-constrained, multi-frontier search, recording experiments in a reproducible registry.
5.  **performance-optimize**: Scientific bottleneck diagnosis and refactoring. Analyzes and refactors code to achieve peak time and memory performance using advanced JAX vectorization, intelligent JIT staging, optimal tensor network contraction, and memory-efficient autodiff strategies.
6.  **tc-rosetta**: Autonomously translates quantum scripts from other frameworks (Qiskit, PennyLane) into TensorCircuit-NG, applying JAX vectorization/JIT and outputting strict before-and-after execution time benchmarking reports.
7.  **tutorial-crafter**: Transforms a raw TensorCircuit-NG script into a comprehensive, self-contained, and narrative-driven tutorial in Markdown/HTML, blending physics/math background, step-by-step code walkthroughs, and HPC programming highlights.
