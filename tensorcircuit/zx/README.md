# ZX-Calculus Module

This module provides support for ZX-calculus based quantum circuit simulation and simplification. It enables efficient sampling of Clifford+T circuits with noise trajectories, leveraging JAX-accelerated backends for high-performance simulation.

Most of the content in this module is adopted from the [tsim](https://github.com/QuEraComputing/tsim) repository (Apache-2.0 license).


## Credits

This module is based on the [tsim](https://github.com/QuEraComputing/tsim) package and the simulation approach described in:
- Haenel, R., Luo, X., & Zhao, C. (2026). "Tsim: Fast Universal Simulator for Quantum Error Correction." *arXiv preprint arXiv:2604.01059*.

## Caution

This module is highly experimental and most of code is directly translated from tsim and generated by generative AI (gemini and gpt), try on your own risks.
